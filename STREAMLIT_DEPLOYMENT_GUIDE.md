## Quick Deployment (5 Minutes)

### Step 1: Go to Streamlit Cloud
Visit: https://streamlit.io/cloud

### Step 2: Sign In
- Click "Sign in"
- Use GitHub account (recommended)
- Authorize Streamlit

### Step 3: Deploy New App
1. Click "New app"
2. Select:
   - **Repository:** `Bhushan-Nanvare/Unemployment-trend-analysis`
   - **Branch:** `main`
   - **Main file:** `app.py`
3. Click "Deploy"

### Step 4: Wait for Deployment
- Takes 2-5 minutes
- You'll see a live URL when ready
- Example: `https://unemployment-intelligence-platform.streamlit.app`

### Step 5: Test the App
1. Click the live URL
2. Test all 11 pages
3. Verify data loads correctly
4. Check interactive features

## Troubleshooting

### If deployment fails:
1. Check GitHub repository is public
2. Verify `app.py` exists in root
3. Check `requirements.txt` has all dependencies
4. Review Streamlit Cloud logs for errors

### If pages show errors:
1. Check internet connection
2. Verify API endpoints are accessible
3. Check data files exist
4. Review browser console for errors

### If data doesn't load:
1. Check World Bank API is accessible
2. Verify job postings CSV exists
3. Check PLFS data file is present
4. Review Streamlit logs
